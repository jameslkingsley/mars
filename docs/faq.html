What is Mars?

Mars is a total Zeus replacement providing Arma communities with a customisable editor capable of realtime mission creation and enhancement.

How is Mars different from Zeus?

- Spawning directly on server or any of the connected headless clients
- Proper interface (like Eden) without clunky module framework
- Far more in-depth AI control via context menus
- Proper 3rd party mod support - entire framework for any kind of UI or function element you can imagine
- More entities visible (those that are visible in Eden but not in Zeus)

Is it compatible with AI mods?

Technically yes, but I would advise not running any AI mods that change behavioural stuff such as ASR AI and VCOM as the mod will have (in the future) its own major AI behaviour changes.

Can I still run Ares and Zeus?

Yes. Additionally Mars has a Zeus login option in the client menu.

Can multiple people be in the Mars editor at the same time?

Yes. Technically you can have an unlimited amount of people in the editor concurrently, however keep in mind that the more people there are spawning things, the more problems you're likely to encounter. I'd recommend no more than 3 users concurrently.

Can players change their view distance and terrain detail?

Yes. Bind the client menu key in CBA controls and use that to do it.

Are there settings for who can access the editor?

Yes. These settings are set on the server via the userconfig. You can whitelist/blacklist by Steam ID and enable first-time login.

Is there headless client support?

Yes. Mars supports an infinite number of headless clients.

Does it work on community made maps?

Yes. It works on every and any map.

Does it show content from all addons, or just vanilla?

All addons. Anything that is in the game's config on game start will be serialized and used in the asset browser.

Is TFAR/ACRE supported?

Only ACRE for now. There is an optional ACRE addon in the optionals folder.

Download doesn't contain any PBO's?

Download the mod from the releases page.

How do I open the Mars editor?

In Configure > Controls > Configure Addons > Mars bind the client menu and editor keys. You can use either to login to the editor.
